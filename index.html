Overview

Umnini Digital Boundaries is a system designed to digitize home boundaries and manage them online. The system is structured into three hierarchical levels: Izizwe (Tribes/Nations), Izigodi (Regions/Wards), and Imizi (Homes/Households).

  • Interactive map for marking property boundaries.
  • Secure portal for homeowners to register and verify land ownership.
  • GPS tracking and document uploads for property validation.
  • Mobile-friendly interface for easy access and updates.
  • Future plans for AI-based land mapping and blockchain security.

User Profiles

Manage profiles for different user roles such as Inkosi (Chief), Induna (Headman), and Homeowner. Each role has specific access permissions and functionalities.

  • Inkosi Profile – Manage Izizwe and see all Izigodi and Imizi under them.
  • Induna Profile – Manage a specific Isigodi and see all Imizi under it.
  • Homeowner Profile – Access their own Imizi boundary and basic Isigodi info.

Search and Update Requests

Users can search for their land by name, ID, or coordinates. If a boundary is incorrect, they can submit a request for updates.

Upload Legal Documentation

Users and leaders can upload documents to verify land ownership. This ensures that all land boundaries are legally recognized.
